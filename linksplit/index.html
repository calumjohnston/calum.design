LinkSplit.io

Visit

Analytics Webtool for A/B testing landing page variations.

Description

Linksplit.io allows you to create an a/b testing URL with two possible destinations. Users visiting this link will be equally split between the two possible desinations. This allows you to A/B test the effectiveness of each variation via your analytics platform.

Technology

  • Node.JS
  • Express Framework
  • MongoDB
  • Pug Templating Engine
  • Deployed via Heroku
