Brisk Table 2017 - Current @ NCH Software

DESCRIPTION

I lead the Front-End Design & Development of Brisk Table, a suite of b2b cloud-based applications currently in development at NCH Software.

NOTABLE FEATURES

Built from the ground up using vanilla JavaScript, BriskTable's Menu Generator (Alpha) allows users to create restaurant menus from scratch.

  • Designed to display consistently across all devices.
  • Image Uploading, utilising LocalStorage API's to allow for instant client-side redering of the image, while it is simultaneously uploaded to the server in the background.
  • The live menu preview is generated client-side using SVG elements. The SVG is regenerated on keyboard input via an array of JavScript SVG Layout functions. text-wrapping, page overs flowing

LinkSplit.io Personal Project 2018 - Current

DESCRIPTION

LinkSplit.io, my current personal project, is an analytics web application designed to simplify the process of A/B testing landing page variations.
Users are able to paste in two URLs, and LinkSplit will create a shortened URL. This shortened URL will split any traffic that visits it between one of the two variations, with a perfect 50/50 split.

NOTABLE FEATURES

  • 3000+ Unique visitors in the first 24hrs from launch, with positive user feedback.
  • Written in JavaScript, using Node JS + Express.
  • If a duplicate URL is detected, the original will be served - No duplicate data.
  • URLs are stored in an mlab.com cloud-hosted MongoDB configuration.
  • Responsive Design using CSS3 Media Queries. Over 50% of the site traffic is from mobile devices.

TECHNOLOGIES

  • HTMl5
  • CSS3
  • Vanilla JavaScript + AJAX
  • Proprietary C++ Server
  • MySQL
