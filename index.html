Brisk Table 2017 - Current @ NCH Software

SCREENSHOTS

DESCRIPTION

Part of my current role involves leading the Front-End Design & Development of Brisk Table, a suite of b2b cloud-based applications currently in development at NCH Software.

A typical workflow for a BriskTable component starts with brain-storming and rough sketches, followed by wireframes and prototypes in Adobe XD. Assets, colours and branding are then imported from our design system before being translated to HTML5, CSS3 and JS.

NOTABLE FEATURES

Built from the ground-up using vanilla JavaScript ES6 (requirement), BriskTable's Menu Generator (Alpha) allows users to create restaurant menus from scratch.

  • Designed to display consistently across all browsers and devices, with an emphasis on accessibility.
  • The image uploading process utilises LocalStorage to allow for instant client-side redering of the thumbnail preview, while it is simultaneously uploaded to the server in the background. This allows for a seamless user-experience.
  • The live menu preview is generated client-side using SVG elements. The SVG is created via an array of JavaScript SVG Text-to-SVG functions, including text-wrapping, page-overflow detection, and margining.
  • Menu Generator is able to function without an internet connection after the initial page load (See Progressive Web Application), due to a Single-Page Application architecture that is designed to simulate the reactivity of native mobile and desktop applications. The only "internet connectivity required" portion of the page is the PDF download, which may also be pushed into the client-side in future.

TECHNOLOGIES

  • HTML5
  • CSS3 + Pre-processors
  • JavaScript + AJAX
  • Proprietary C++ Server
  • MySQL

LinkSplit.io Personal Project 2018 - Current

SCREENSHOTS

DESCRIPTION

LinkSplit.io, my current personal project, is an analytics web application designed to simplify the process of A/B testing landing page variations.
Users are able to paste in two URLs, and LinkSplit will create a shortened URL. This shortened URL will split any traffic evenly between the two variations, allowing the user to evaluate which of the two variations was more effective without the need to integrate an A/B Testing Framework.

NOTABLE FEATURES

  • Fully responsive Design using CSS3 Media Queries. Over 50% of the site traffic is from mobile devices.
  • 3000+ Unique visitors in the first 24 hours from launch, with very positive user feedback.
  • Written in JavaScript, using Node JS + Express.
  • If a duplicate URL is detected, the original will be served resulting in no duplicate data.
  • URLs are stored in an mlab.com Cloud-Hosted MongoDB configuration.

TECHNOLOGIES

  • NodeJS + Express
  • Pug Node Templating Engine
  • CSS3 + SASS
  • jQuery
  • MongoDB Cloud-Hosted Database
